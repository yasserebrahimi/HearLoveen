# ApiGateway

## Description
YARP API Gateway with OAuth 2.0

## Port
5000

## Technology Stack
- .NET 8
- Entity Framework Core 8
- MediatR (CQRS)
- FluentValidation
- Serilog
- Health Checks
- Swagger/OpenAPI

## Architecture
- Clean Architecture
- CQRS Pattern
- Domain-Driven Design
- Repository Pattern
- Unit of Work Pattern

## Endpoints
See Swagger documentation at http://localhost:5000/swagger

## Getting Started
```bash
cd src/ApiGateway
dotnet restore
dotnet run
```
