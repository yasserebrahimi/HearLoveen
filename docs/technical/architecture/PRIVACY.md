# PRIVACY.md (GDPR/Kids)

- **Lawful basis**: parental consent (Article 8). Special care for children (Recital 38).
- **Data minimization**: keep derived features and reports; limit raw audio retention (e.g. 30–90 days).
- **EU residency**: process/store data in EU regions only.
- **DSR**: export (machine‑readable), erasure with cascading delete, time‑boxed responses.
- **No ads / no sale of data**. No speaker‑ID unless explicit purpose and consent.
